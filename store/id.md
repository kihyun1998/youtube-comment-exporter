Cara paling sederhana untuk mengekspor komentar YouTube ke CSV atau JSON.

Pernah ingin menganalisis atau menyimpan komentar dari video YouTube, tapi terlalu banyak untuk disalin secara manual?
YouTube Comment Exporter memungkinkan Anda mengunduh semua komentar dan balasan dari video mana pun hanya dalam beberapa klik.

━━━━━━━━━━━━━━━━━━━━
Untuk siapa ini?
━━━━━━━━━━━━━━━━━━━━
• Kreator — Tinjau umpan balik penonton secara ringkas di spreadsheet
• Marketer & Peneliti — Kumpulkan data komentar untuk analisis tren atau sentimen
• Pengarsip — Cadangkan komentar berharga sebelum video dihapus

━━━━━━━━━━━━━━━━━━━━
Fitur Utama
━━━━━━━━━━━━━━━━━━━━
✔ Ekspor CSV / JSON — Pilih format yang Anda butuhkan dari dropdown sederhana
✔ Balasan Termasuk — Ambil semua balasan bersarang secara inline, tidak ada yang terlewat
✔ Pengambilan Paralel Cepat — 8x permintaan bersamaan untuk pengumpulan massal yang cepat
✔ Deteksi Otomatis ID Video — Klik ikon di tab YouTube mana pun dan ID terisi otomatis
✔ Dukungan Tempel URL — Tempel URL YouTube lengkap dan ID diekstrak otomatis
✔ Bilah Progres — Bilah progres visual dengan persentase, batalkan kapan saja
✔ Ekspor Terpisah — Pisahkan file ekspor berdasarkan utas komentar untuk analisis lebih mudah
✔ Nama File Kustom — Sesuaikan nama file ekspor dengan template fleksibel
✔ Tema Gelap / Terang — Beralih dengan satu klik untuk tampilan nyaman
✔ 16 Bahasa — Tersedia dalam Bahasa Indonesia, Inggris, Jepang, Cina, Korea, dan lainnya

━━━━━━━━━━━━━━━━━━━━
Cara Penggunaan
━━━━━━━━━━━━━━━━━━━━

1. Instal ekstensi dan masukkan kunci YouTube Data API Anda di Pengaturan.
2. Buka video YouTube mana pun dan klik ikon ekstensi.
3. ID Video terisi otomatis. Pilih CSV atau JSON dan tekan Ekspor.
4. File Anda terunduh. Selesai!

※ Diperlukan kunci YouTube Data API v3. Anda bisa mendapatkannya secara gratis dari Google Cloud Console.
