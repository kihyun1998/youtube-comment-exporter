كيفية الاستخدام
━━━━━━━━━━━━━━━━━━━━

1. ثبّت الإضافة وأدخل مفتاح YouTube Data API في الإعدادات.
2. انتقل إلى أي فيديو YouTube وانقر على أيقونة الإضافة.
3. يتم ملء معرف الفيديو تلقائيًا. اختر CSV أو JSON واضغط على تصدير.
4. يتم تنزيل ملفك. تم!

※ يلزم مفتاح YouTube Data API v3. يمكنك الحصول عليه مجانًا من Google Cloud Console.
