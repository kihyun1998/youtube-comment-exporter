将YouTube评论导出为CSV或JSON的最简单方法。

是否曾想分析或保存YouTube视频的评论，但数量太多无法手动复制？
YouTube Comment Exporter让您只需几次点击即可下载所有评论和回复。

使用方法
━━━━━━━━━━━━━━━━━━━━

1. 安装扩展程序，在设置中输入YouTube Data API密钥。
2. 前往任意YouTube视频，点击扩展程序图标。
3. 视频ID自动填入。选择CSV或JSON，点击导出。
4. 文件下载完成！

※ 需要YouTube Data API v3密钥。可从Google Cloud Console免费获取。
