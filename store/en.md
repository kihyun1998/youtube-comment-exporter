The simplest way to export YouTube comments to CSV or JSON.

Ever wanted to analyze or save comments from a YouTube video, but there were just too many to copy by hand?
YouTube Comment Exporter lets you download every comment and reply from any video in just a few clicks.

Key Features
━━━━━━━━━━━━━━━━━━━━
✔ CSV / JSON Export — Pick the format you need from a simple dropdown
✔ Replies Included — Fetch all nested replies inline, nothing left behind
✔ Fast Parallel Fetching — 8x concurrent requests for speedy bulk collection
✔ Auto-detect Video ID — Click the icon on any YouTube tab and the ID fills in automatically
✔ URL Paste Support — Paste a full YouTube URL and the ID is extracted for you
✔ Real-time Progress — Watch the progress live and cancel anytime
✔ Dark / Light Theme — One-click toggle for comfortable viewing
✔ 16 Languages — Available in English, Korean, Japanese, Chinese, Spanish, and more

How to Use
━━━━━━━━━━━━━━━━━━━━

1. Install the extension and enter your YouTube Data API key in Settings.
2. Go to any YouTube video and click the extension icon.
3. The Video ID is filled in automatically. Choose CSV or JSON and hit Export.
4. Your file downloads. Done!

※ A YouTube Data API v3 key is required. You can get one for free from the Google Cloud Console.
