YouTubeのコメントをCSVまたはJSONでエクスポートする最も簡単な方法。

YouTube動画のコメントを分析・保存したいのに、多すぎて手作業ではコピーしきれなかったことはありませんか？
YouTube Comment Exporterを使えば、数クリックですべてのコメントと返信をダウンロードできます。

こんな方におすすめ
━━━━━━━━━━━━━━━━━━━━
• クリエイター — スプレッドシートで視聴者のフィードバックを一目で確認
• マーケター・研究者 — トレンドや感情分析のためのコメントデータ収集
• アーカイバー — 動画が削除される前に貴重なコメントをバックアップ

━━━━━━━━━━━━━━━━━━━━
主な機能
━━━━━━━━━━━━━━━━━━━━
✔ CSV / JSONエクスポート — シンプルなドロップダウンから必要な形式を選択
✔ 返信も含む — すべてのネストされた返信をインラインで取得
✔ 高速並列取得 — 8倍の同時リクエストで大量収集を高速化
✔ 動画ID自動検出 — YouTubeタブでアイコンをクリックするとIDが自動入力
✔ URL貼り付け対応 — YouTube URLを貼り付けるとIDが自動抽出
✔ リアルタイム進捗 — 進捗をリアルタイムで確認し、いつでもキャンセル可能
✔ ダーク / ライトテーマ — ワンクリックで切り替え
✔ 16言語対応 — 日本語、英語、韓国語、中国語、スペイン語など

━━━━━━━━━━━━━━━━━━━━
使い方
━━━━━━━━━━━━━━━━━━━━

1. 拡張機能をインストールし、設定でYouTube Data APIキーを入力します。
2. YouTube動画に移動し、拡張機能アイコンをクリックします。
3. 動画IDが自動入力されます。CSVまたはJSONを選択してエクスポートをクリック。
4. ファイルがダウンロードされます。完了！

※ YouTube Data API v3キーが必要です。Google Cloud Consoleから無料で取得できます。
