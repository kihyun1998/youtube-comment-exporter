YouTube टिप्पणियों को CSV या JSON में निर्यात करने का सबसे सरल तरीका।

क्या आपने कभी किसी YouTube वीडियो की टिप्पणियों का विश्लेषण या सेव करना चाहा, लेकिन वे इतनी ज़्यादा थीं कि हाथ से कॉपी नहीं कर सकते थे?
YouTube Comment Exporter से आप किसी भी वीडियो की सभी टिप्पणियाँ और उत्तर बस कुछ क्लिक में डाउनलोड कर सकते हैं।

यह किसके लिए है?
━━━━━━━━━━━━━━━━━━━━
• क्रिएटर्स — स्प्रेडशीट में दर्शकों की प्रतिक्रिया एक नज़र में देखें
• मार्केटर्स और शोधकर्ता — ट्रेंड या सेंटिमेंट विश्लेषण के लिए टिप्पणी डेटा एकत्र करें
• आर्काइवर्स — वीडियो हटाए जाने से पहले मूल्यवान टिप्पणियों का बैकअप लें

प्रमुख विशेषताएँ
━━━━━━━━━━━━━━━━━━━━
✔ CSV / JSON निर्यात — एक सरल ड्रॉपडाउन से आवश्यक प्रारूप चुनें
✔ उत्तर शामिल — सभी नेस्टेड उत्तर इनलाइन प्राप्त करें, कुछ भी छूटे नहीं
✔ तेज़ समानांतर फ़ेचिंग — तेज़ बल्क संग्रह के लिए 8x समवर्ती अनुरोध
✔ वीडियो ID स्वतः पहचान — किसी भी YouTube टैब पर आइकन क्लिक करें और ID स्वतः भर जाती है
✔ URL पेस्ट सपोर्ट — पूरा YouTube URL पेस्ट करें और ID स्वतः निकाली जाती है
✔ प्रगति बार — प्रतिशत ट्रैकिंग के साथ विज़ुअल प्रगति बार, कभी भी रद्द करें
✔ विभाजित निर्यात — आसान विश्लेषण के लिए टिप्पणी थ्रेड्स के अनुसार फ़ाइलें विभाजित करें
✔ कस्टम फ़ाइल नाम — लचीले टेम्पलेट्स से निर्यात फ़ाइल नाम अनुकूलित करें
✔ डार्क / लाइट थीम — आरामदायक देखने के लिए वन-क्लिक टॉगल
✔ 16 भाषाएँ — हिंदी, अंग्रेज़ी, जापानी, चीनी, कोरियाई और अन्य में उपलब्ध

उपयोग कैसे करें
━━━━━━━━━━━━━━━━━━━━

1. एक्सटेंशन इंस्टॉल करें और सेटिंग्स में अपनी YouTube Data API कुंजी दर्ज करें।
2. किसी भी YouTube वीडियो पर जाएँ और एक्सटेंशन आइकन पर क्लिक करें।
3. वीडियो ID स्वतः भर जाती है। CSV या JSON चुनें और निर्यात पर क्लिक करें।
4. आपकी फ़ाइल डाउनलोड हो जाती है। हो गया!

※ YouTube Data API v3 कुंजी आवश्यक है। आप Google Cloud Console से मुफ़्त में प्राप्त कर सकते हैं।
