A maneira mais simples de exportar comentários do YouTube para CSV ou JSON.

Já quis analisar ou salvar comentários de um vídeo do YouTube, mas eram muitos para copiar manualmente?
O YouTube Comment Exporter permite baixar todos os comentários e respostas de qualquer vídeo em poucos cliques.

━━━━━━━━━━━━━━━━━━━━
Para quem é?
━━━━━━━━━━━━━━━━━━━━
• Criadores — Revise o feedback dos espectadores de forma rápida em uma planilha
• Marketers e pesquisadores — Colete dados de comentários para análise de tendências ou sentimentos
• Arquivadores — Faça backup de comentários valiosos antes que um vídeo seja removido

Principais recursos
━━━━━━━━━━━━━━━━━━━━
✔ Exportação CSV / JSON — Escolha o formato necessário em um menu suspenso simples
✔ Respostas incluídas — Obtenha todas as respostas aninhadas inline, sem deixar nada para trás
✔ Busca paralela rápida — 8x requisições simultâneas para coleta em massa veloz
✔ Detecção automática do ID do vídeo — Clique no ícone em qualquer aba do YouTube e o ID é preenchido automaticamente
✔ Suporte a colagem de URL — Cole uma URL completa do YouTube e o ID é extraído automaticamente
✔ Progresso em tempo real — Acompanhe o progresso ao vivo e cancele a qualquer momento
✔ Tema escuro / claro — Alternância com um clique para visualização confortável
✔ 16 idiomas — Disponível em português, inglês, japonês, chinês, coreano e mais

━━━━━━━━━━━━━━━━━━━━
Como usar
━━━━━━━━━━━━━━━━━━━━

1. Instale a extensão e insira sua chave da YouTube Data API nas Configurações.
2. Acesse qualquer vídeo do YouTube e clique no ícone da extensão.
3. O ID do vídeo é preenchido automaticamente. Escolha CSV ou JSON e clique em Exportar.
4. Seu arquivo é baixado. Pronto!

※ É necessária uma chave da YouTube Data API v3. Você pode obter uma gratuitamente no Google Cloud Console.
