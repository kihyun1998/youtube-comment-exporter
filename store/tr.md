YouTube yorumlarını CSV veya JSON olarak dışa aktarmanın en basit yolu.

Bir YouTube videosunun yorumlarını analiz etmek veya kaydetmek istediniz ama elle kopyalanamayacak kadar çok mu vardı?
YouTube Comment Exporter, herhangi bir videonun tüm yorumlarını ve yanıtlarını sadece birkaç tıkla indirmenizi sağlar.

━━━━━━━━━━━━━━━━━━━━
Kimler için?
━━━━━━━━━━━━━━━━━━━━
• İçerik üreticileri — İzleyici geri bildirimlerini bir elektronik tabloda hızlıca inceleyin
• Pazarlamacılar ve araştırmacılar — Trend veya duygu analizi için yorum verilerini toplayın
• Arşivciler — Video kaldırılmadan önce değerli yorumları yedekleyin

Temel Özellikler
━━━━━━━━━━━━━━━━━━━━
✔ CSV / JSON Dışa Aktarma — Basit bir açılır menüden ihtiyacınız olan formatı seçin
✔ Yanıtlar Dahil — Tüm iç içe yanıtları satır içi olarak alın, hiçbir şey atlanmaz
✔ Hızlı Paralel Getirme — Hızlı toplu toplama için 8x eşzamanlı istek
✔ Otomatik Video ID Algılama — Herhangi bir YouTube sekmesinde simgeye tıklayın, ID otomatik doldurulur
✔ URL Yapıştırma Desteği — Tam YouTube URL'sini yapıştırın, ID otomatik çıkarılır
✔ İlerleme Çubuğu — Yüzde takibi ile görsel ilerleme çubuğu, istediğiniz zaman iptal edin
✔ Bölünmüş Dışa Aktarma — Daha kolay analiz için yorum dizilerine göre dosyaları bölün
✔ Özel Dosya Adı — Esnek şablonlarla dışa aktarma dosya adını özelleştirin
✔ Koyu / Açık Tema — Rahat görüntüleme için tek tıkla geçiş
✔ 16 Dil — Türkçe, İngilizce, Japonca, Çince, Korece ve daha fazlasında kullanılabilir

Nasıl Kullanılır
━━━━━━━━━━━━━━━━━━━━

1. Uzantıyı yükleyin ve Ayarlar'da YouTube Data API anahtarınızı girin.
2. Herhangi bir YouTube videosuna gidin ve uzantı simgesine tıklayın.
3. Video ID otomatik doldurulur. CSV veya JSON seçin ve Dışa Aktar'a basın.
4. Dosyanız indirilir. Tamam!

※ YouTube Data API v3 anahtarı gereklidir. Google Cloud Console'dan ücretsiz olarak alabilirsiniz.
