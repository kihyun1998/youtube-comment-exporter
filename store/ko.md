YouTube 댓글을 CSV 또는 JSON으로 내보내는 가장 간단한 방법.

YouTube 영상의 댓글을 분석하거나 저장하고 싶었지만, 너무 많아서 일일이 복사할 수 없었던 적 있으신가요?
YouTube Comment Exporter를 사용하면 클릭 몇 번으로 모든 댓글과 답글을 다운로드할 수 있습니다.

━━━━━━━━━━━━━━━━━━━━
누구를 위한 확장 프로그램인가요?
━━━━━━━━━━━━━━━━━━━━
• 크리에이터 — 스프레드시트에서 시청자 피드백을 한눈에 확인
• 마케터 & 연구자 — 트렌드 또는 감성 분석을 위한 댓글 데이터 수집
• 아카이버 — 영상이 삭제되기 전에 소중한 댓글 백업

━━━━━━━━━━━━━━━━━━━━
주요 기능
━━━━━━━━━━━━━━━━━━━━
✔ CSV / JSON 내보내기 — 간단한 드롭다운에서 원하는 형식 선택
✔ 답글 포함 — 모든 중첩 답글을 인라인으로 가져오기
✔ 빠른 병렬 수집 — 8배 동시 요청으로 대량 수집 속도 향상
✔ 비디오 ID 자동 감지 — YouTube 탭에서 아이콘을 클릭하면 ID가 자동 입력
✔ URL 붙여넣기 지원 — YouTube URL을 붙여넣으면 ID가 자동 추출
✔ 실시간 진행률 — 진행 상황을 실시간으로 확인하고 언제든 취소 가능
✔ 다크 / 라이트 테마 — 원클릭 토글로 편안한 화면
✔ 16개 언어 지원 — 한국어, 영어, 일본어, 중국어, 스페인어 등 지원

━━━━━━━━━━━━━━━━━━━━
사용 방법
━━━━━━━━━━━━━━━━━━━━

1. 확장 프로그램을 설치하고 설정에서 YouTube Data API 키를 입력하세요.
2. YouTube 영상으로 이동한 후 확장 프로그램 아이콘을 클릭하세요.
3. 비디오 ID가 자동으로 입력됩니다. CSV 또는 JSON을 선택하고 내보내기를 누르세요.
4. 파일이 다운로드됩니다. 완료!

※ YouTube Data API v3 키가 필요합니다. Google Cloud Console에서 무료로 발급받을 수 있습니다.
