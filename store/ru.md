Самый простой способ экспортировать комментарии YouTube в CSV или JSON.

Когда-нибудь хотели проанализировать или сохранить комментарии к видео на YouTube, но их было слишком много, чтобы копировать вручную?
YouTube Comment Exporter позволяет загрузить все комментарии и ответы к любому видео всего в несколько кликов.

━━━━━━━━━━━━━━━━━━━━
Для кого это?
━━━━━━━━━━━━━━━━━━━━
• Создатели контента — Просматривайте отзывы зрителей в таблице одним взглядом
• Маркетологи и исследователи — Собирайте данные комментариев для анализа трендов или настроений
• Архивисты — Сохраняйте ценные комментарии до того, как видео будет удалено

━━━━━━━━━━━━━━━━━━━━
Основные функции
━━━━━━━━━━━━━━━━━━━━
✔ Экспорт в CSV / JSON — Выберите нужный формат из простого выпадающего меню
✔ Ответы включены — Получайте все вложенные ответы встроенно, ничего не пропуская
✔ Быстрая параллельная загрузка — 8x одновременных запросов для быстрого массового сбора
✔ Автоопределение ID видео — Нажмите иконку на любой вкладке YouTube, и ID заполнится автоматически
✔ Вставка URL — Вставьте полный URL YouTube, и ID будет извлечён автоматически
✔ Прогресс в реальном времени — Следите за прогрессом в прямом эфире и отменяйте в любой момент
✔ Тёмная / Светлая тема — Переключение одним кликом для комфортного просмотра
✔ 16 языков — Доступно на русском, английском, японском, китайском, корейском и других

━━━━━━━━━━━━━━━━━━━━
Как использовать
━━━━━━━━━━━━━━━━━━━━

1. Установите расширение и введите ключ YouTube Data API в Настройках.
2. Перейдите на любое видео YouTube и нажмите иконку расширения.
3. ID видео заполняется автоматически. Выберите CSV или JSON и нажмите Экспорт.
4. Ваш файл загружается. Готово!

※ Требуется ключ YouTube Data API v3. Вы можете получить его бесплатно в Google Cloud Console.
