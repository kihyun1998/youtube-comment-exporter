Le moyen le plus simple d'exporter les commentaires YouTube en CSV ou JSON.

Vous avez déjà voulu analyser ou sauvegarder les commentaires d'une vidéo YouTube, mais ils étaient trop nombreux pour les copier à la main ?
YouTube Comment Exporter vous permet de télécharger tous les commentaires et réponses de n'importe quelle vidéo en quelques clics.

À qui s'adresse-t-il ?
━━━━━━━━━━━━━━━━━━━━
• Créateurs — Consultez les retours des spectateurs d'un coup d'œil dans un tableur
• Marketeurs et chercheurs — Collectez les données de commentaires pour l'analyse de tendances ou de sentiments
• Archivistes — Sauvegardez les commentaires précieux avant qu'une vidéo ne soit supprimée

Fonctionnalités principales
━━━━━━━━━━━━━━━━━━━━
✔ Export CSV / JSON — Choisissez le format dont vous avez besoin dans un menu déroulant simple
✔ Réponses incluses — Récupérez toutes les réponses imbriquées en ligne, rien n'est laissé de côté
✔ Récupération parallèle rapide — 8x requêtes simultanées pour une collecte en masse rapide
✔ Détection automatique de l'ID vidéo — Cliquez sur l'icône dans n'importe quel onglet YouTube et l'ID se remplit automatiquement
✔ Collage d'URL — Collez une URL YouTube complète et l'ID est extrait automatiquement
✔ Barre de progression — Barre de progression visuelle avec pourcentage, annulez à tout moment
✔ Export divisé — Divisez les fichiers d'export par fils de commentaires pour une analyse plus facile
✔ Nom de fichier personnalisé — Personnalisez le nom du fichier d'export avec des modèles flexibles
✔ Thème sombre / clair — Basculement en un clic pour un affichage confortable
✔ 16 langues — Disponible en français, anglais, japonais, chinois, coréen et plus

Comment utiliser
━━━━━━━━━━━━━━━━━━━━

1. Installez l'extension et entrez votre clé YouTube Data API dans les Paramètres.
2. Accédez à n'importe quelle vidéo YouTube et cliquez sur l'icône de l'extension.
3. L'ID de la vidéo est rempli automatiquement. Choisissez CSV ou JSON et cliquez sur Exporter.
4. Votre fichier se télécharge. Terminé !

※ Une clé YouTube Data API v3 est requise. Vous pouvez en obtenir une gratuitement depuis la Google Cloud Console.
