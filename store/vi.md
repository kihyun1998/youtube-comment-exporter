Cách đơn giản nhất để xuất bình luận YouTube sang CSV hoặc JSON.

Bạn đã bao giờ muốn phân tích hoặc lưu bình luận từ video YouTube, nhưng quá nhiều để sao chép thủ công chưa?
YouTube Comment Exporter cho phép bạn tải xuống mọi bình luận và phản hồi từ bất kỳ video nào chỉ trong vài cú nhấp chuột.

Tính năng chính
━━━━━━━━━━━━━━━━━━━━
✔ Xuất CSV / JSON — Chọn định dạng bạn cần từ menu thả xuống đơn giản
✔ Bao gồm phản hồi — Lấy tất cả phản hồi lồng nhau inline, không bỏ sót
✔ Tải song song nhanh — 8x yêu cầu đồng thời để thu thập hàng loạt nhanh chóng
✔ Tự động phát hiện ID Video — Nhấp vào biểu tượng trên bất kỳ tab YouTube nào và ID được điền tự động
✔ Hỗ trợ dán URL — Dán URL YouTube đầy đủ và ID được trích xuất tự động
✔ Thanh tiến trình — Thanh tiến trình trực quan với phần trăm, hủy bất cứ lúc nào
✔ Xuất chia tách — Chia tệp xuất theo luồng bình luận để phân tích dễ dàng hơn
✔ Tên tệp tùy chỉnh — Tùy chỉnh tên tệp xuất với mẫu linh hoạt
✔ Giao diện tối / sáng — Chuyển đổi một cú nhấp để xem thoải mái
✔ 16 ngôn ngữ — Có sẵn bằng tiếng Việt, Anh, Nhật, Trung, Hàn và nhiều ngôn ngữ khác

Cách sử dụng
━━━━━━━━━━━━━━━━━━━━

1. Cài đặt tiện ích mở rộng và nhập khóa YouTube Data API trong Cài đặt.
2. Truy cập bất kỳ video YouTube nào và nhấp vào biểu tượng tiện ích mở rộng.
3. ID Video được điền tự động. Chọn CSV hoặc JSON và nhấn Xuất.
4. Tệp của bạn được tải xuống. Xong!

※ Cần có khóa YouTube Data API v3. Bạn có thể lấy miễn phí từ Google Cloud Console.
