La forma más sencilla de exportar comentarios de YouTube a CSV o JSON.

¿Alguna vez quisiste analizar o guardar los comentarios de un video de YouTube, pero eran demasiados para copiar a mano?
YouTube Comment Exporter te permite descargar todos los comentarios y respuestas de cualquier video en solo unos clics.

━━━━━━━━━━━━━━━━━━━━
¿Para quién es?
━━━━━━━━━━━━━━━━━━━━
• Creadores — Revisa los comentarios de tus espectadores de un vistazo en una hoja de cálculo
• Marketers e investigadores — Recopila datos de comentarios para análisis de tendencias o sentimientos
• Archivadores — Haz una copia de seguridad de comentarios valiosos antes de que se elimine un video

━━━━━━━━━━━━━━━━━━━━
Características principales
━━━━━━━━━━━━━━━━━━━━
✔ Exportación CSV / JSON — Elige el formato que necesitas desde un menú desplegable
✔ Respuestas incluidas — Obtén todas las respuestas anidadas en línea, sin dejar nada atrás
✔ Obtención paralela rápida — 8x solicitudes concurrentes para una recopilación masiva veloz
✔ Detección automática del ID de video — Haz clic en el icono en cualquier pestaña de YouTube y el ID se completa automáticamente
✔ Soporte para pegar URL — Pega una URL completa de YouTube y el ID se extrae automáticamente
✔ Progreso en tiempo real — Observa el progreso en vivo y cancela en cualquier momento
✔ Tema oscuro / claro — Cambio con un solo clic para una visualización cómoda
✔ 16 idiomas — Disponible en español, inglés, japonés, chino, coreano y más

━━━━━━━━━━━━━━━━━━━━
Cómo usar
━━━━━━━━━━━━━━━━━━━━

1. Instala la extensión e ingresa tu clave de YouTube Data API en Configuración.
2. Ve a cualquier video de YouTube y haz clic en el icono de la extensión.
3. El ID del video se completa automáticamente. Elige CSV o JSON y presiona Exportar.
4. Tu archivo se descarga. ¡Listo!

※ Se requiere una clave de YouTube Data API v3. Puedes obtener una gratis en Google Cloud Console.
