將YouTube留言匯出為CSV或JSON的最簡單方法。

是否曾想分析或儲存YouTube影片的留言，但數量太多無法手動複製？
YouTube Comment Exporter讓您只需幾次點擊即可下載所有留言和回覆。

使用方法
━━━━━━━━━━━━━━━━━━━━

1. 安裝擴充功能，在設定中輸入YouTube Data API金鑰。
2. 前往任意YouTube影片，點擊擴充功能圖示。
3. 影片ID自動填入。選擇CSV或JSON，點擊匯出。
4. 檔案下載完成！

※ 需要YouTube Data API v3金鑰。可從Google Cloud Console免費取得。
