
### 상세 설명 (Full Description)
```
Hey Product Hunt! 👋

I'm excited to launch YouTube Comment Exporter — a free, open-source
browser extension that makes exporting YouTube comments effortless.

## The Problem
If you've ever tried to analyze or save YouTube comments, you know
the pain. There's no built-in export feature, and copying hundreds
(or thousands) of comments manually is not realistic.

## The Solution
YouTube Comment Exporter lets you download every comment and reply
from any YouTube video with just a few clicks.

## Key Features
🔹 **One-click export** — CSV or JSON, your choice
🔹 **All replies included** — Nested replies fetched automatically
🔹 **8x parallel fetching** — Fast even on videos with thousands of comments
🔹 **Real-time progress** — Visual progress bar with percentage, cancel anytime
🔹 **Split export** — Break files by comment threads for easier analysis
🔹 **Custom filenames** — Flexible filename templates
🔹 **16 languages** — EN, KO, JA, ZH, ES, PT, DE, FR, HI, and more
🔹 **Dark & Light theme** — Comfortable viewing in any environment

## Who is this for?
- **Content Creators** — Review all viewer feedback in a spreadsheet
- **Marketers & Researchers** — Collect comment data for sentiment analysis
- **Data Archivists** — Back up comments before videos disappear

## Tech Stack
Built with React 19, TypeScript, WXT, Tailwind CSS 4, and Zustand.
Fully open source on GitHub.

I'd love to hear your feedback and suggestions! 🙏
```
